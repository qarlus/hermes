# Iris

Iris is the working name for the embeddable Hermes terminal platform.

The name is intentionally short and easy to remember. It fits the same messenger lineage as Hermes, but it stands on its own as a terminal technology brand inside the repo.

Why `Iris`:

- short enough to use in paths, package names, and internal discussions
- linked to messaging and movement without reusing Hermes itself
- suggests rendering, focus, and visual quality, which matches the terminal goal
- distinct enough to avoid sounding like a generic SSH utility

This folder captures the product and engineering direction for Iris as an internal platform, not a standalone application.

Naming stack:

- `Hermes`: the overall product
- `Iris`: the embeddable terminal platform
- `imux`: the integrated workspace shell/window that hosts terminals, git, browser, and adjacent operator surfaces
