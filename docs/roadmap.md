# Hermes Roadmap

## Phase 0: Foundation

- Establish Bun workspaces and shared package boundaries.
- Stand up Tauri v2 desktop shell with Vite/React.
- Define Nox design tokens and desktop interaction patterns.

## Phase 1: Desktop Local MVP

- SQLite-backed host management.
- Searchable host list with favorites.
- Terminal tabs with xterm.js.
- PTY-backed system `ssh` connections.
- Remote tmux attach-or-create flow.

## Later phases

- Secure credential storage and OS-native keychain integration.
- Encrypted sync and self-hosted sync service.
- SFTP and file workflows.
- Mobile client using shared core models.
